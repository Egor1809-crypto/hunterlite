# План перехода HUNTERLITE к модульному монолиту

Цель: превратить текущий frontend MVP в полноценную SaaS-платформу с понятными модулями, backend, базой данных, ролями, безопасностью, тестами и подготовкой к production.

## Текущее состояние

Сейчас проект — это frontend MVP-монолит на React/Vite:

- интерфейс платформы уже собран;
- основные страницы и роли визуально представлены;
- данные лежат в frontend-моках;
- авторизация пока визуальная;
- backend, база данных, настоящие роли и защищенные маршруты еще не реализованы;
- есть unit smoke-тесты, lint и production build.

## Целевая архитектура

На ближайшем этапе строим модульный монолит:

```text
apps/web                  frontend-приложение
apps/api                  backend API
packages/shared           общие типы, схемы, константы
modules/auth              вход, сессии, OAuth, восстановление пароля
modules/users             пользователи, профили, статусы
modules/organizations     компании/тенанты, участники, изоляция данных
modules/roles             роли и права доступа
modules/trainings         тренировки, сценарии, ответы, оценки
modules/exams             экзамены, допуски, результаты
modules/notifications     уведомления, шаблоны, доставка
modules/admin             админка, настройки, аудит действий
modules/analytics         отчеты, динамика баллов, слабые темы
modules/ai                AI-клиент, разбор ответов, prompt версии
infra                     Docker, окружения, деплой
docs                      архитектура, решения, инструкции
```
